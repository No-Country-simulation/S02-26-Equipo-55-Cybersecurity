# 👥 Guía de Concientización en Seguridad
**Super App Security Kit – v1.0**  
**Clasificación:** Uso Interno  

---

# 1. Objetivo

Fortalecer la cultura de seguridad dentro del equipo y reducir el riesgo de incidentes causados por error humano.

La seguridad es responsabilidad de todos.

---

# 2. Principales Amenazas para el Equipo

## 2.1 Phishing

Correos o mensajes fraudulentos que intentan obtener:

- Credenciales
- Información financiera
- Acceso a sistemas internos

### Señales de alerta:

- Urgencia excesiva
- Errores ortográficos
- Enlaces sospechosos
- Dominios similares pero incorrectos

---

## 2.2 Ingeniería Social

Manipulación psicológica para obtener información confidencial.

Ejemplos:

- Llamadas fingiendo ser soporte técnico
- Solicitudes de acceso urgente
- Mensajes de "alta dirección"

---

## 2.3 Uso inseguro de dispositivos

- Conectarse a WiFi públicas sin VPN
- Compartir dispositivos de trabajo
- Instalar software no autorizado

---

# 3. Buenas Prácticas Obligatorias

- Activar autenticación multifactor.
- No compartir credenciales.
- Usar gestor de contraseñas.
- Bloquear pantalla al ausentarse.
- Reportar actividades sospechosas inmediatamente.

---

# 4. Procedimiento ante sospecha de ataque

1. No interactuar con el contenido sospechoso.
2. Tomar captura de pantalla.
3. Notificar al responsable técnico.
4. Cambiar contraseña si fue expuesta.
5. Documentar el incidente.

---

# 5. Simulaciones y Capacitación

Se recomienda:

- Simulaciones internas de phishing cada 6 meses.
- Capacitación anual obligatoria.
- Actualización constante ante nuevas amenazas.

---

# 6. Cultura de Seguridad

La seguridad no es un obstáculo para el negocio, es un habilitador de confianza.

Cada miembro del equipo es la primera línea de defensa.
